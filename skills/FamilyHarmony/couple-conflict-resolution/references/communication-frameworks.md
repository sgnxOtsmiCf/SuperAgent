# 夫妻沟通框架与案例分析

## 框架一：戈特曼"爱情实验室"原则

### 四大杀手（需避免）
1. **批评**：针对人格的攻击（"你就是个懒人"）
2. **蔑视**：优越感、嘲讽、翻白眼
3. **防御**：推卸责任、找借口（"都是因为你..."）
4. ** stonewalling（筑墙）**：拒绝沟通，冷处理

### 修复尝试信号
当一方发出和解信号（如自嘲、软化语气），另一方应积极回应，而非继续攻击。

## 框架二：情绪聚焦疗法（EFT）

### 恶性循环识别
- **追逐者**：焦虑、要求、指责 → 对方逃避
- **逃避者**：退缩、沉默、冷漠 → 对方更焦虑

### 破解方法
- 追逐者：表达脆弱而非愤怒（"我很害怕失去你"）
- 逃避者：承认并表达感受（"我确实感到压力"）

## 实战对话模板

### 场景：财务分歧

**错误示范**：
A: "你乱花钱，我们永远存不下钱！"
B: "你管得太宽了，我自己赚的钱！"

**正确示范**：
A: "看到账单时我很焦虑（感受），因为我需要财务安全感（需求）。我们可以一起制定预算计划吗（请求）？"
B: "我理解你的担心（共情）。我也希望有积蓄（共同目标）。关于个人消费空间，我们能否商量一个双方都能接受的额度（合作）？"

## 自我反思清单

### 给提问者的问题：
1. 我在这段关系中真正害怕失去的是什么？
2. 我的反应有多少来自过去的创伤（原生家庭）？
3. 如果一年后回头看，我希望自己如何处理这次冲突？
4. 我是否把伴侣当成了敌人，而非队友？
