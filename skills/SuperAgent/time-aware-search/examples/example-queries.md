# 时间感知搜索技能 - 示例查询

## 示例 1：询问最新新闻

**用户输入**：最近有什么人工智能方面的新闻？

**系统处理流程**：
1. 检测时间敏感词："最近"、"新闻"
2. 获取当前时间：2026-04-16 14:30:00
3. 计算时间范围：2026-04-09 至 2026-04-16（最近7天）
4. 构建搜索查询："人工智能新闻 2026年 最新"
5. 调用搜索工具获取结果
6. 生成时效性回答

**预期回答示例**：
> 现在是2026年4月16日，我为您搜索了近期的相关新闻：
> 
> 根据科技日报（2026-04-15）的报道，XXX公司发布了新一代AI模型...
> 
> 健康时报（2026-04-14）也提到人工智能在医疗领域取得突破...

## 边界情况示例

### 情况 1：多个时间词汇
**输入**："今年和去年相比，人工智能有什么新进展？"
- 检测词汇："今年"、"去年"、"新"、"进展"
- 处理：分别处理两个时间段，对比分析

### 情况 2：隐含时间需求
**输入**："OpenAI 现在怎么样了？"
- 检测词汇："现在"
- 处理：获取当前时间，搜索最新动态

### 情况 3：未来时间
**输入**："明年人工智能会有什么新趋势？"
- 检测词汇："明年"、"趋势"
- 处理：基于当前时间（2026年），搜索"2027年人工智能预测"
